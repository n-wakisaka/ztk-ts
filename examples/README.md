# Examples

このディレクトリには、`ztk-ts` の主要 API を人が読んで理解するためのサンプルを置く。

- `api-basics.ts`
  - `parseZtk()`
  - `resolveZtk()`
  - `serializeZtk()`
  - `serializeZtkNormalized()`
  - `serializeSemanticZtkPreservingSource()`
  - `serializeSemanticZtkNormalized()`
- `materialized-runtime.ts`
  - `analyzeSemanticZtkMaterializedRuntime()`
  - `serializeSemanticZtkMaterializedRuntime()`
  - import format 判定と current diagnostic の読み方

どちらもテストより説明を優先しており、コメントは日本語で丁寧に書いてある。
